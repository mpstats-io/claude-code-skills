# Claude Code Skills — MPSTATS

Скиллы для AI-агентов с полным справочником MPSTATS API. После установки агент сам работает с аналитикой маркетплейсов — без написания кода.

## Установка

```bash
npx skills add https://github.com/mpstats-io/claude-code-skills -y -g
```

Работает с Claude Code, Cursor, Codex, Windsurf, Cline и другими агентами.

## Скиллы

### mpstats

Полный справочник MPSTATS API для аналитики Wildberries, Ozon и Яндекс Маркет.

**122 эндпоинта, 30+ готовых скриптов.**

Что умеет агент:
- Анализ категорий, ниш, брендов, продавцов — продажи, выручка, тренды, ценовая сегментация
- SKU-аналитика: история продаж, остатки по складам и размерам, позиции в поиске, отзывы
- Сравнение периодов, похожие товары, сезонность
- Генерация аналитических отчётов в фирменном стиле MPSTATS

**Требуется токен MPSTATS API.** Получить в личном кабинете: [mpstats.io/userpanel](https://mpstats.io/userpanel) (блок "API token"). Токен передаётся агенту через переменную окружения `MPSTATS_TOKEN`.

## Как устроен скилл

```
mpstats/
├── SKILL.md          — точка входа, индекс
├── scripts/          — bash-скрипты для API-вызовов
├── references/       — детальные API-контракты
└── config/           — настройки
```

`SKILL.md` загружается в контекст агента как индекс. Скрипты из `scripts/` агент вызывает напрямую. Файлы из `references/` подгружаются по необходимости — контекст не забивается, но доступ к любому методу API есть.
