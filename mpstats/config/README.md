# MPSTATS: получение и сохранение токена

## Шаг 1: Получите токен в MPSTATS

1. Откройте [mpstats.io](https://mpstats.io/) и войдите в аккаунт.
2. Перейдите в пользовательскую панель: [https://mpstats.io/userpanel](https://mpstats.io/userpanel).
3. В блоке API token скопируйте ваш персональный токен (его можно скопировать прямо в кабинете).

Важно:
- Для доступа к API нужна активная подписка MPSTATS.
- Тарифы и актуальные акции: [https://mpstats.io/tariffs](https://mpstats.io/tariffs).

Важно:
- Токен может быть сброшен при смене пароля.
- Токен меняется после ручной регенерации в кабинете.

## Шаг 2: Сохраните токен в скилле (постоянно для этого workspace)

В корне скилла `mpstats` выполните:

```bash
cp config/.env.example config/.env
```

Откройте `config/.env` и вставьте значение:

```env
MPSTATS_TOKEN=ваш_токен_здесь
```

После этого все скрипты в `scripts/` будут автоматически подхватывать токен из `config/.env`.

## Шаг 3: Проверка

```bash
bash scripts/account-limits.sh
```

Если токен корректный, вернется ответ по лимитам API.

## Приоритет источников токена

1. `MPSTATS_TOKEN` из переменной окружения текущей команды.
2. `MPSTATS_TOKEN` из `config/.env`.

Это позволяет временно переопределять токен без изменения файла.
